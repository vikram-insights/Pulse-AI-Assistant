🚀 Pulse AI Assistant

A modular command-line AI Assistant built using Python that integrates real-world APIs and utility tools. This project is designed to strengthen Python fundamentals, API integration, modular programming, error handling, and software development practices.

---

✨ Features (v1.0)

- 📅 Current Date & Time
- 🌦️ Live Weather Information
- ₿ Live Bitcoin Price
- 🧮 Calculator
- 💱 Currency Converter
- 🌐 Website Launcher (including Custom URLs)

---

🛠️ Tech Stack

- Python 3
- Requests Library
- Open-Meteo API
- Exchange Rate API
- CoinGecko API
- Webbrowser Module
- Git & GitHub

▶️ Getting Started

Clone the repository

git clone <repository-url>

Go to project folder

cd Pulse

Install dependencies

pip install requests

Run the project

python main.py

📸 Features

📅 Date & Time

Displays the current system date and time.

🌦️ Weather

Fetches live weather information for any city.

₿ Bitcoin Price

Displays the latest Bitcoin price using a public API.

🧮 Calculator

Supports:

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Power

💱 Currency Converter

- Live exchange rates
- Multiple currencies
- Automatic USD bridge conversion

🌐 Website Launcher

- Opens predefined websites
- Supports custom URLs
- Automatically handles missing https://
